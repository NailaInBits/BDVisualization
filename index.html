Tracking the Bengali Diaspora with D3.js

This is a passion project started by Nishat Anjum. The citations for the data used in this project can be found here. Unfortunately, a lot of the data collected in the decade after the Bengali Independence is severely enqueued, you can read more about it here. Therefore take the data presented with a grain of salt and keep in mind that more often than not when people note down population trends they start with which populations can cause the most disruption which historically has been the male gender. The data from 1990 onward is sourced from the United Nations.
